========================================
  闲鱼商品智能分析系统 - 快速使用指南
========================================

📋 安装步骤：

1. 安装Python依赖：
   pip install -r requirements.txt

2. 安装Playwright浏览器：
   playwright install chromium

3. 启动服务器：
   方式1：双击 start.bat 文件
   方式2：进入backend目录，运行 python app.py

4. 打开浏览器访问：
   http://localhost:5000

========================================

🚀 使用流程：

1. 在界面中输入您的 Gemini API Key
   （如果没有，请访问 https://makersuite.google.com/app/apikey 获取）

2. 输入要搜索的商品关键词
   例如：iPhone 15 Pro、笔记本电脑、显卡等

3. 点击"开始搜索与分析"按钮

4. 系统会弹出浏览器窗口
   ⚠️ 首次使用需要登录闲鱼账号
   ✅ 登录状态会自动保存，下次无需重复登录

5. 等待系统自动：
   - 爬取商品数据
   - 使用 Gemini AI 分析商品性价比
   - 显示推荐结果

========================================

💡 注意事项：

- 首次使用需要在弹出的浏览器窗口中手动登录闲鱼
- 登录状态保存在 browser_state.json 文件中
- 如果使用代理访问 Gemini API，请确保代理运行在 http://127.0.0.1:7890
- 爬虫会显示浏览器窗口，方便查看和调试

========================================

❓ 常见问题：

Q: 提示"找不到爬虫脚本"？
A: 请确保 crawler_enhanced.py 文件在项目根目录

Q: 无法连接 Gemini API？
A: 检查代理设置，确保可以访问 Google 服务

Q: 爬虫没有数据？
A: 检查是否已登录闲鱼账号，或尝试手动刷新浏览器窗口

========================================
